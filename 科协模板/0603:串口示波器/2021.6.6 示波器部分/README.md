# 用Python编写的带GUI的简易串口示波器
仅实现了波形图的绘制，没有实现频谱图绘制
支持自动寻找可用串口号
仅用如下语句进行了简单的模拟串口输出:
```c
 for (int i=0;i<100;i++)
      {
          sprintf(show,"%.2f\r",3.3*sin(i*PI2/100));
          print(show);
          delay_ms(1);
      }
```
没有用实际的ADC数据进行测试

串口读取过程中有部分数字会被截断，造成后面数据读取和处理的错误。
此问题并未解决，仅仅是根据输出数据特点对其进行了清洗。
